Это отборочные задания на смену от Т-банка в Новосибирской области.
файл first - решение первого задания. 
На ввод поступают 2 числа через пробел, начало и конец диапазона соответсвенно.
файл second - решение второго задания.
В коде есть комментарии поясняющие смысл некоторых функций. При работе с меню следует использовать цифры и кириллицу. 
При добавлении различной информации, например, название или описание книг, можно использовать любой язык.
Все данные сохраняются в файл library.json.
